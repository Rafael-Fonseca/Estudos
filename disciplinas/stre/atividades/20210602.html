Atividade 01

Pesquise sobre outra linguagem de descrição de hardware, chamada Verilog, compare com VHDL:
Resposta:
Verilog é uma linguagem fracamente tipada, é menos deterministica que VHDL, possui sintaxe semelhante a linguagem C.
Verilog é menos verbosa que VHDL. Mas no tocante à descrição de hardware essas diferenças não importam.

Atividade 02

a) Quais são as duas principais estruturas de projeto do VHDL?
b) Defina as diferenças e funções destas duas estruturas.
c) O exemplo apresentado anteriormente mostrava a descrição VHDL de uma porta AND.
Agora escreva o código para a porta OR de duas entradas, sabendo que em VHDL a porta or é definida como OR.
Respostas:
a)
Entidade e Arquitetura.
b)
Uma arquiterura pertence a uma entidade.
Uma entidade só possui uma arquitetura.
Na entidade declaramos as interfaces do sistema com o mundo real:
Estas interfaces podem ser dos tipos, entrada, saída, buffer e inout.
Saídas não podem ser lidas, Buffer é uma saída que pode ser lida. E inout pode ser in ou out dependendo da impedância.
Nas arquiteturas é onde informamos a lógica do sistema. Uma arquitetura pode fazer uso de outras entidades.
c)
Ignore os simbolos de "_"
entity circuito is
____port( a, b, c: in bit;
_________s_______: out bit
____);
end entity;

architecture estrutura of circuito is
____begin
________s <= a or b;
____end architecture;
