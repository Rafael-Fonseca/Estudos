Como funciona a alocação de memória em Python?


Em python este processo é realizado pelo Python Memory Manager.
Python é implementado utilizando a linguagem C e este processo ocorre no modo C.
Python possui um coletor de lixo embutido, ele utiliza a técnica contagem de referência, nela, um objeto é desalocado da memória quando o programa não possui mais referências a ele.
Desta forma, o desenvolvedor python não precisa se preocupar em criar um coletor de lixo.
Preocupação esta que deve ter um desenvolvedor da linguagem C, que nativamente não implementa um coletor de lixo.
