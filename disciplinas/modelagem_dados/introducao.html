Introdução:

    Neste estágio do livro são apresentados conceitos básicos de banco de dados que nos permitem entender melhor as ideias descritas no decorrer do livro.
    • Redundância de dados:
      •  Como o nome sugere, é a situação onde a mesma informação se encontra repitida em um SI.
      Existem dois tipos de redundância de dados, a controlada e a não controlada.
    • Redundãcia de dados controlada:
      •  É a redundância de dados em que o próprio SI garante que as informações repetidas tenham o mesmo valor.
      Por exemplo: Considere que um sistema guarda informações sobre a quantidade de um determinado produto, em duas partes distintas do sistema. E ocorre a venda de uma unidade deste produto, então o software assegura que o valor seja atualizado nas duas partes do sistema.
    • Redundãcia de dados não controlada:
      •  A responsabilidade de atualizar os dados em todos os pontos de repetição é do usuário do sistema.
      Este tipo de redundância deve ser evitado, uma vez que sua implementação gera problemas ao usuário como redigitação e incosistência de dados.
    • Banco de Dados(BD):
      •  É um conjunto de dados que objetiva atender seus usuários.
    • Sitema de Gerência de Banco de Dados(SGBD):
      •  Software que permite criar, ler e atualizar dados em um BD.
      O autor não traz a funcionalidade deletar dados em um BD, mas na minha convicção pessoal, eu levo.
      Se eu fosse você leitor, ficaria com a definição do autor rs.
    • Modelo de dados:
      •  É descrição formal da estrutura de um BD. Pode ser realizado de duas formas, conceitual ou lógico.
    • Modelo Conceitual:
      •  É um modelo abstrato que descreve um banco de dados sem levar em conta nenhum SGBD em particular.
      A técnica mais usada para montar modelos conceituais é a abordagem Entidade Relacionamento(ER), através de um Diagrama Entidade Relacionamento(DER).
    • Modelo Lógico:
      •  É uma descrição de um banco de dados no nível visto pelo usuário do SGBD. Ou seja, este modelo precisa levar em consideração qual SGBD será utilizado.
