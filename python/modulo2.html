  • PyPi(pip):
    •  Empacotador de bibliotecas que possibilita reaproveitar código de forma correta e compartilhada com a comunidade python.

  • Pytest:
    •  Biblioteca que possibilita criar testes.

  • Coverage:
    •  Biblioteca que mostra a porcentagem de código que está coberta por testes.

  • Test Driven Development:
    •  Desenvolvimento dirigido a testes(TDD) é uma forma de desenvolver código, onde os testes são realizados antes
    dos códigos. O desenvolvimento ocorre em passos, onde cada passo é dado de forma a passar no próximo teste.

  • Baby Steps:
    •  Completa o TDD, fazendo com que cada passo seja realizado da maneira mais simples possível, tanto para a criação
    do teste, quanto para a sua solução.
