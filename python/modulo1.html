Aqui não será exposta nenhuma parte do curso. Será um índice remissivo de novidades que eu aprendi com o curso.

  • Escrita de números inteiros:
    •  1000000(Um milhão) pode ser escrito como 1_000_000. E dessa forma é muito mais legível.
  • Definição de strings com '''(3 aspas simples):
    •  Serve para utilizar aspas simples e duplas em uma mesma string e também para saltar linhas sem utilizar \n.
  • dir(objeto):
    •  Função que retorna todas as funções e atributos do objeto passado como parâmetro.
  • Desempacotamento de tupla:
    •  Uma vez que eu tenha declarado uma tupla como nome_tupla = (param_1, param_2)
    Posso informar: var_1, var_2 = nome_tupla
    O sistema fará var_1 = param_1 e var_2 = param_2.
  • Inverter com fatiamento:
    •  variável[::-1], serve para strings, listas e tuplas.
  • enumerate(iteravel):
    •  Documentação
    Retorna uma lista de tuplas onde[(indice, elemento), ...,] ambos do parâmetro iteravel.
  • PEP 8:
    •  Esse eu sabia mas vou anotar assim mesmo.
    Nome de variáveis e funções minuscula_com_underline.
  • Parâmetros variáveis:
    •  Por justaposição: def função(*args) onde *args será uma tupla.
    Por parâmetros nomeados: def função(**kwargs) onde **kwargs será um dicionário.
    Podem ser usadas em conjunto: def função(*args, **kwargs).
    Caso já tenha uma tupla ou dicionário pré definido, o mesmo pode ser passado como parâmetro mas é necessário realizar o desempacotamento,
    a menos quê, você deseje, ter a tupla ou dicionário em questão como um único indice do seu args ou kwargs.
    Para desempacotar basta informar o nome da variável com os asteriscos, ou seja, no mesmo modelo da definição.
